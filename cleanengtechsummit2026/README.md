# CLEANENGTECHSUMMIT2026

Official website for the Global Summit on Clean Energy and Sustainable Technologies

## Event Details

- **Date:** March 8–10, 2027
- **Subdomain:** cleanengtechsummit2026.sciengasummits.com
- **Theme:** Building a Sustainable Future Through Clean Energy Innovation

## About

A premier international summit bringing together leading experts, researchers, policymakers, and industry innovators in clean energy and sustainable technologies from around the world.

## Tech Stack

- React 18.2.0
- React Router DOM 6.20.0
- Vite 5.0.8
- Plain CSS (no frameworks)

## Features

- Multi-page navigation with React Router
- Countdown timer to event date
- Interactive speaker cards with modal popups
- Responsive design for all devices
- Professional green/sustainability color scheme
- High-quality Unsplash images
- Demo forms (non-functional backend)

## Project Structure

```
src/
├── components/
│   ├── Navbar.jsx/css          # Sticky navigation
│   ├── Footer.jsx/css          # Footer with links
│   ├── CountdownTimer.jsx/css  # Event countdown
│   └── SpeakerCard.jsx/css     # Speaker cards with modals
├── pages/
│   ├── Home.jsx/css            # Landing page with hero
│   ├── About.jsx/css           # Summit overview
│   ├── Tracks.jsx/css          # Conference tracks
│   ├── Speakers.jsx/css        # Featured speakers
│   ├── SubmitAbstract.jsx/css  # Abstract submission
│   ├── Register.jsx/css        # Registration tiers
│   ├── Sponsors.jsx/css        # Sponsorship packages
│   ├── Venue.jsx/css           # Venue information
│   └── Contact.jsx/css         # Contact form & FAQ
├── App.jsx                     # Main app with routing
├── App.css                     # Global app styles
├── index.css                   # CSS variables & base styles
└── main.jsx                    # Entry point
```

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
