# CLEANENGTECHSUMMIT2026 - Setup Guide

## Quick Start

```bash
# Navigate to project directory
cd powerengsummit2026/cleanengtechsummit2026

# Install dependencies (if not already installed)
npm install

# Start development server
npm run dev

# Open browser to http://localhost:5173
```

## Project Overview

A premium, sustainability-focused conference website for the Global Summit on Clean Energy and Sustainable Technologies (March 8-10, 2027).

## Features Implemented

✅ **Multi-page React application** with React Router
✅ **9 fully functional pages:**
   - Home (with hero, countdown, impact metrics)
   - About (vision, mission, why attend)
   - Tracks (6 conference tracks with topics)
   - Speakers (4 featured speakers with modal bios)
   - Submit Abstract (demo form)
   - Register (4 ticket tiers)
   - Sponsors (3 sponsorship packages)
   - Venue (coming soon page)
   - Contact (form + FAQ)

✅ **Professional Components:**
   - Sticky responsive navbar
   - Countdown timer to event date
   - Speaker cards with modal popups
   - Footer with links
   - All fully responsive

✅ **Design:**
   - Green/sustainability color palette
   - High-quality Unsplash images
   - Smooth animations and transitions
   - Mobile-first responsive design
   - No AI-generated placeholder boxes

✅ **Images from Unsplash:**
   - Wind turbines and solar panels
   - Sustainable cities
   - Clean energy infrastructure
   - Professional speaker portraits

## Build for Production

```bash
npm run build
```

Output will be in the `dist/` folder, ready for deployment.

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile browsers (iOS Safari, Chrome Mobile)

- Images load from Unsplash CDN
- Fully client-side application
